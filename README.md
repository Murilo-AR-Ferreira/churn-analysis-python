#Churn Analysis - Cancelamento de Clientes

#Objetivo:
Analisar os fatores que influenciam o cancelamento de clientes (churn) e identificar possíveis ações para reduzir a taxa de cancelamento.

#Tecnologias utilizadas:
- Python
- Pandas
- Plotly
- Jupyter Notebook

#Etapas da análise:
1. Importação e tratamento dos dados
2. Remoção de valores nulos
3. Cálculo da taxa geral de cancelamento
4. Análise da influência de:
   - Duração do contrato
   - Número de ligações ao call center
5. Simulação de estratégias para redução do churn

#Principais Insights:

- Clientes com contrato mensal apresentam maior taxa de cancelamento.
- Clientes que realizaram mais de 4 ligações ao call center têm maior probabilidade de cancelar.
- Aplicando melhorias nesses pontos, a taxa de churn pode ser reduzida significativamente.
